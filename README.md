# ⚡ JSync Subscription Engine

A lightweight **Java-based SaaS Subscription & Billing System** that simulates real-world payment platforms like Netflix, Spotify, and SaaS billing engines.

It supports both:
- 💳 Flexible one-time billing (user-defined duration)
- 🔁 Subscription-based auto-renewal system

---

## 🚀 Features

### 💰 Billing System
- Dynamic bill calculation based on plan + months
- Discount engine (tier-based pricing)
- Balance validation & payment processing

### 🔁 Subscription Engine
- Auto-renewal using billing cycles
- Next billing date tracking
- Active / Inactive subscription states

### 🧠 Smart Logic
- Low balance detection
- Graceful failure handling
- State-based system design
